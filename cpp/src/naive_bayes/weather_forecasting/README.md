# Naive Bayes Weather Forecasting (C++)

This module implements a Naive Bayes classifier for a simple weather forecasting dataset.
The focus is on computing prior probabilities, likelihoods, and using them to make predictions.

---

## Objective

- Compute prior probabilities for the target variable
- Compute likelihoods for feature values conditioned on each class
- Use priors and likelihoods to predict the most likely forecast
